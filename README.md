# Flappy Bird Clone (Python & Pygame)
Here is a breakdown of what I implemented in this project: 
<li><strong>Object-Oriented Programming (OOP):</strong> Structured the entire codebase using reusable classes.</li>
<li><strong>Game Physics & Mechanics:</strong> Coded gravity-based movement, real-time collision detection, and automated obstacle generation.</li>
<li><strong>State Management:</strong> Built a scene-based architecture to handle transitions seamlessly (Menu, Play, Pause, and Game Over).</li>
<li><strong>File I/O & Audio:</strong> Added local file saving/loading to track high scores across sessions, alongside dynamic audio integration.</li>
